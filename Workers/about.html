WORKERS ATTENDANCE SYSTEM

Quick-Sign

Quick-sign is a system build for automating the processing of attendance, this will enhance the speed of performing attendance task easily. The system will also help in evacuating attendance eligibility criteria of an employee, it will also falicitate to access the attendance information of aparticular employee in an organisation.

Here is our User Guide.

The website is supposed to allow workers to easily check-in, check-out and also view shared documents in their computers.

Admin(HR)

  1. The admin creates account and add employees into the attendance system.
  2. Admin issues employees with username and password where the default username and password are employee number and identity number respectively.
  3. When the admin logs-in he/she can view employee check-in and check-out, and post the notifications to the employees.

User(EMPLOYEE)

The user now logs-in to the system where he or she is required to check-in, check-out and might view notifications such as memos. When done he or she has the liberty to log -out.
